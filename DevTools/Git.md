# Configs
```sh
git config --global user.name “[firstname lastname]”
git config --global user.email “[valid-email]”
git config --global color.ui auto

```

---

# Setting new Repository
```sh

git clone [url]
```

```sh
get add .
```

# Branch & Merge

List all branches
```sh
git branch 
```

```sh

git branch world
```

```sh
git checkout world
```
